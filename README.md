# visual-editor
基于vue-cli，使用vue3.2 setup语法糖结构和TypeScript编写、pinia作为状态管理库、sortablejs实现元素拖拽、element-plus作为UI库。
## 安装依赖
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
